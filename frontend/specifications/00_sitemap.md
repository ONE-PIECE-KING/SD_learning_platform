## 圖例說明

| 顏色/符號 | 意義 | 說明 |
| --------- | ---- | ---- |
| 藍色實心 | Page（頁面） | 獨立路由的完整頁面 |
| 淺藍色 | Component / Section | 頁面內的區塊元件 |
| 灰色 | Elements | UI 元素（按鈕、輸入框等） |
| 白色 | Description | 描述性文字或備註 |
| 橘黃色 | 特殊標記頁面 | 需特殊處理的頁面 |
| 深色/黑色 | 暫緩實作 | 標記為暫不開發的功能 |

## 全域元件 (Global Components)

### Header Section（全站導覽列）

所有頁面共用的頂部導覽列，提供站內頁面快速跳轉。

| 導覽項目 | 連結目標 | 備註 |
| -------- | -------- | ---- |
| Logo / 首頁 | `/` | 點擊 Logo 回首頁 |
| 課程總覽 | `/courses` | |
| 一對一諮詢 | `/consult` | CTA 強調樣式 |
| 資源分享 | `/resources` | |
| 註冊 / 登入 | `/auth/login` | 未登入時顯示 |
| 會員中心 | `/dashboard` | 已登入時顯示（取代註冊/登入） |
| 通知鈴鐺 | — | 已登入時顯示，下拉面板組件（詳見 `11_notification_component.md`） |
| 購物車 | `/cart` | 圖示 + 數字徽章 |

### Footer Section（全站頁腳）

所有頁面共用的底部區塊。

| 區塊 | 內容 |
| ---- | ---- |
| 聯絡資訊 | Email、電話、營業時間等 |
| 社群連結 | YouTube、Facebook、Instagram、LinkedIn、Discord |
| 法律連結 | 隱私權政策（`/legal/privacy`）、退費政策（`/legal/refund`）、服務條款（`/legal/terms`） |
| 版權聲明 | Copyright 年份、公司名稱 |

## 各頁面結構摘要

> 以下為各頁面的架構摘要。詳細元件規格、API 整合、交互行為等請參閱各獨立規格文件。

### 1. 首頁 Home (`/`)

品牌形象建立的核心頁面，由上至下引導訪客完成轉換。詳見 `01_homepage.md`。

| 順序 | 區塊 | 目的 | 核心 CTA |
| ---- | ---- | ---- | -------- |
| 1 | Hero Section | 品牌主視覺，第一印象 | 免費領取轉職戰略手冊（Lead Magnet） |
| 2 | 信任徽章條 | 建立專業信任感 | — |
| 3 | 主推課程 | 展示精選課程卡片，激發興趣 | 查看更多課程 |
| 4 | 主要服務 | 介紹一對一諮詢與模擬面試 | 立即預約諮詢 |
| 5 | 痛點與解決方案 | 引發共鳴並提供解方 | — |
| 6 | 社會證明 | 學員見證與成果展示 | — |
| 7 | Final CTA | 最終轉換行動呼籲 | 免費領取轉職手冊 / 瀏覽所有課程 |

---

### 2. 課程總覽 (`/courses`)

展示所有課程的列表頁，支援搜尋、篩選、排序與分頁。詳見 `03_course_overview.md`。

| 區塊 | 說明 |
| ---- | ---- |
| 頂部搜尋區 | 搜索列 + 分類標籤 / 麵包屑導航 |
| 側邊篩選欄 | 類別、難易度、價格區間、評分篩選 |
| 主要內容區 | 排序控制 + 課程卡片網格 + 分頁控制 |

---

### 3. 課程詳細資訊 (`/courses/:id`)

單一課程的完整介紹頁面，含側邊購買懸浮欄。詳見 `04_course_detail.md`。

| 區塊 | 說明 |
| ---- | ---- |
| Hero Section | 課程基本資訊 + 媒體預覽窗 |
| 側邊購買懸浮欄 | Sticky 購買卡片（未購買/已購買/講師身分切換 CTA） |
| 核心內容分頁 | 課程簡介、課程大綱（手風琴）、講師簡介、學員評價 |

---

### 4. 一對一諮詢 (`/consult`)

諮詢預約工具頁面，提供導師資訊瀏覽、方案選擇、時段預約與需求填寫。詳見 `05_consultation.md`。

| 區塊 | 說明 |
| ---- | ---- |
| 導師個人檔案區 | 頭像、姓名、專業標籤、諮詢統計、個人簡介 |
| 諮詢方案選擇 | 時長、價格、諮詢方式 |
| 預約時段選取器 | 日曆視圖 + 時段列表 + 時區切換 |
| 諮詢需求表單 | 諮詢主題、附件上傳、預期目標 |
| 社群連結 | LINE、Discord 即時聯繫按鈕 |

---

### 5. 資源分享 (`/resources`)

文章部落格系統，提供免費學習資源與知識分享。詳見 `06_resources.md`。

| 分類 | 內容方向 |
| ---- | -------- |
| 技術新訊 | 業界技術動態與趨勢 |
| 知識分享 | 資料科學相關教學文章 |
| 提示詞工程 | AI Prompt Engineering 教學與範例 |
| 轉職案例 | 學員轉職資料科學的真實故事 |

## 頁面導航流程

### 主要使用者旅程

```
[訪客] ─── 首頁 ─┬─ Hero CTA ──────── Email 訂閱（Lead Magnet）
                  ├─ 搜尋 ──────────── 課程總覽
                  ├─ 主推課程卡片 ───── 課程詳細資訊 ─── 加入購物車 ─── 購物車 ─── 結帳
                  ├─ 主要服務 ─────── 一對一諮詢 ───── 註冊帳號 ───── 預約
                  └─ Final CTA ────── 註冊 / 訂閱電子報

[會員] ─── 登入 ─┬─ 個人頁 / 學生上課頁
                  ├─ 購物車 → 結帳
                  └─ 課程學習
```

### 頁面間跳轉關係

| 起始頁面 | 目標頁面 | 觸發方式 |
| -------- | -------- | -------- |
| 首頁 | 課程總覽 | Header 導覽 / SearchBar / 查看更多課程 |
| 首頁 | 一對一諮詢 | Header 導覽 / Main Services CTA |
| 首頁 | 登入/註冊 | Header 導覽 / Final CTA 註冊 |
| 課程總覽 | 課程詳細資訊 | 課程卡片點擊 |
| 課程詳細資訊 | 購物車 | CTA 按鈕（加入購物車） |
| 一對一諮詢 | 登入/註冊 | CTA 按鈕（註冊帳號 / 成為老師） |
| 登入/註冊 | 會員中心 | 登入成功自動導向 `/dashboard` |
| Header | 任意頁面 | 全域導覽列 |
