# 法律資訊 頁面規格書

## 頁面資訊

| 項目     | 說明                                                   |
| -------- | ------------------------------------------------------ |
| 路徑     | `/legal/privacy`、`/legal/refund`、`/legal/terms`      |
| 角色權限 | 公開頁面，所有訪客皆可瀏覽                             |
| 核心目的 | 提供隱私權政策、退費政策、服務條款等法律必要資訊       |

---

## 頁面結構

| 區塊             | 說明                                                     |
| ---------------- | -------------------------------------------------------- |
| 導覽選單         | 桌面端左側 / 行動端頂部 Tab 切換                         |
| 內文容器         | 單欄設計 max-width 800px，確保長文閱讀舒適               |
| 快速跳轉目錄     | 右側 Table of Contents，點擊跳轉至特定章節               |

### 通用元件

| 元件           | 說明                                       |
| -------------- | ------------------------------------------ |
| 最後更新日期   | 顯示於標題下方                             |
| 列印按鈕       | 允許存為 PDF 或列印                        |
| 聯絡客服連結   | 底部提供聯絡管道                           |

---

## 元件規格

### 畫面一：隱私權政策（Privacy Policy）

| 章節             | 內容                                                             |
| ---------------- | ---------------------------------------------------------------- |
| 資料收集範圍     | 註冊資訊、交易資訊（不儲存完整信用卡號）、學習紀錄             |
| 資料使用目的     | 提供服務、身分驗證、課程更新通知、行銷推廣（需同意）           |
| Cookie 政策      | 登入狀態紀錄與使用者體驗優化                                   |
| 第三方分享       | 金流服務商（綠界、藍新）、數據分析工具（Google Analytics）     |
| 資料安全與權利   | 加密技術說明、使用者查閱/修改/刪除個資權利                     |

### 畫面二：退費政策（Refund Policy）

**退費基準：**

| 條件             | 退費方式                             |
| ---------------- | ------------------------------------ |
| 未觀看           | 購買後 X 日內且完全未觀看，全額退費 |
| 部分觀看         | 進度低於 X% 且 Y 日內，比例退費     |
| 不予退費         | 進度超過 X% 或超過 Y 日             |

**特殊商品：**
- 實體教材一經寄出/拆封之扣除標準
- 直播課程與錄播課程退費差異

**退費程序：**
- 申請管道：線上表單或客服信箱
- 審核天數：7 個工作天內
- 退款方式：原路退回信用卡或匯款銀行帳戶（手續費說明）

### 畫面三：服務條款（Terms of Service）

| 章節             | 內容                                                       |
| ---------------- | ---------------------------------------------------------- |
| 使用者資格       | 年齡限制、帳戶資訊真實性義務                               |
| 帳戶管理         | 禁止轉讓、禁止多人共用                                     |
| 智慧財產權       | 課程影片/教材版權歸平台或講師，禁止錄影/非法下載/轉載     |
| 行為準則         | 禁止攻擊性、廣告或違法言論                                 |
| 服務中斷         | 不可抗力或系統維護之補償原則                               |
| 法律管轄         | 臺灣臺北地方法院                                           |

## API 整合

```
GET /api/v1/legal/{type}
```

- `type`: `privacy` | `refund` | `terms`
- 回傳格式：Markdown 或 HTML

---

## 狀態與錯誤處理

| 狀態     | 處理方式                         |
| -------- | -------------------------------- |
| Loading  | 內文區域使用 Skeleton Screen     |
| Error    | 顯示「內容載入失敗」+ 重試按鈕  |
