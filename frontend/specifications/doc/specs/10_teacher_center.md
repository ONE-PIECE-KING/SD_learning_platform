# 老師中心 頁面規格書

## 頁面資訊

| 項目     | 說明                                                     |
| -------- | -------------------------------------------------------- |
| 路徑     | `/dashboard/*`（與學生中心共用框架，依角色動態顯示側邊欄項目） |
| 角色權限 | 需登入，角色為老師（部分頁面學生/老師共用）             |
| 核心目的 | 課程建立與管理、銷售數據分析、學生互動管理               |

---

## 頁面結構

### 全局元件

**Header：**
- Logo（點擊返回首頁 `/`）
- 導航連結：課程總覽、一對一諮詢、資源分享
- 右側按鈕：會員中心（`/dashboard`）、通知鈴鐺（下拉面板，詳見 `11_notification_component.md`）、購物車（圖示 + 數字徽章）

**側邊導航欄（Nav）：**

| 項目                 | 身份     | 路徑                       |
| -------------------- | -------- | -------------------------- |
| 購課記錄             | 學生     | `/dashboard/history`       |
| 我的課程             | 學生     | `/dashboard/my-courses`    |
| 課程上架             | 老師     | `/dashboard/course-upload` |
| 統計/分析            | 老師     | `/dashboard/statistics`    |
| 個人簡介             | 共用     | `/dashboard/profile`       |
| 老師聯絡簿           | 老師     | `/dashboard/contact`       |
| 設定                 | 共用     | `/dashboard/settings`      |
| 一對一諮詢           | 共用     | `/dashboard/consult`       |
| 資源分享             | 共用/老師| `/dashboard/resources`     |

---

## 元件規格

### 2.1 購課記錄（學生身份共用）

| 元件       | 說明                                         |
| ---------- | -------------------------------------------- |
| 篩選器     | 消費明細 / 退款明細 / 平台交易明細（選配）  |
| 列表       | 表格：日期、課程名稱、金額、狀態             |

### 2.2 我的課程（學生身份共用）

| 元件               | 說明                                                       |
| ------------------ | ---------------------------------------------------------- |
| 課程列表           | 課程卡片（圖片、標題、簡介、進度條），點擊進入上課頁面     |
| 上課頁面           | 影片播放器 + 討論串 + 筆記 + 聯絡簿 + 章節列表             |

### 2.3 課程上架（`/dashboard/course-upload`）

| 元件               | 說明                                                 |
| ------------------ | ---------------------------------------------------- |
| 課程大綱編輯       | 課程表單：名稱、簡介、封面圖、目標受眾等             |
| 影片上傳區         | 建立章節與單元，上傳各單元教學影片                   |
| 定價編輯區         | 設定售價與優惠方案                                   |
| 送出審核區         | 完成設定後提交平台審核                               |

### 2.4 統計/分析（`/dashboard/statistics`）

| 元件                 | 說明                                             |
| -------------------- | ------------------------------------------------ |
| 儀表板               | 本月購課折線圖、各課程銷售量與收入統計表         |
| 已上架課程列表       | 列出所有已上架課程，提供「更新課程」按鈕         |
| 課程上架捷徑         | 快速前往課程上架頁面                             |

### 2.5 個人簡介（共用）

| 元件           | 說明                                       |
| -------------- | ------------------------------------------ |
| 個人簡介表單   | 頭像、姓名、專業領域、經歷、社群連結       |
| 發布與預覽     | 提供「發布」與「預覽」功能                 |
| 我的證書       | 上傳並展示專業證照                         |
| 我的作品       | 個人作品集或專案連結                       |

### 2.6 老師聯絡簿（`/dashboard/contact`）

| 元件               | 說明                                                   |
| ------------------ | ------------------------------------------------------ |
| 已上架課程選擇     | 選擇要管理的課程                                       |
| 聯絡簿編輯         | 進入管理介面：回覆影片討論串、批改作業、發布測驗、傳送訊息 |

### 2.7 設定（共用）

| 元件                   | 說明                                     |
| ---------------------- | ---------------------------------------- |
| 諮詢功能開關           | 老師選擇是否開放諮詢服務                 |
| Nav 編輯               | 自訂側邊導航項目順序或顯示與否           |

### 2.8 一對一諮詢（共用）

| 元件         | 說明                                                     |
| ------------ | -------------------------------------------------------- |
| 行事曆       | 老師設定可預約時段；學生查看並預約/取消                  |
| 訪談室       | 視訊框、對話框、計時器、計費狀態                         |

### 2.9 資源分享（共用/老師）

| 元件         | 說明                                                     |
| ------------ | -------------------------------------------------------- |
| 文章撰寫     | Rich text editor，文章分類選擇，提交審核/直接發布       |
| 筆記分享     | （選配）學生公開分享課程筆記                             |

## API 整合

| 功能             | 端點                                     | 方法    |
| ---------------- | ---------------------------------------- | ------- |
| 課程建立         | `/api/v1/courses`                        | POST    |
| 課程編輯         | `/api/v1/courses/{id}`                   | PUT     |
| 影片上傳         | `/api/v1/courses/{id}/videos`            | POST    |
| 課程審核提交     | `/api/v1/courses/{id}/submit`            | POST    |
| 銷售統計         | `/api/v1/teacher/statistics`             | GET     |
| 學生互動管理     | `/api/v1/teacher/courses/{id}/contacts`  | GET     |
| 作業批改         | `/api/v1/teacher/homework/{id}`          | PUT     |
| 諮詢時段設定     | `/api/v1/consultants/{id}/availability`  | PUT     |
| 文章 CRUD        | `/api/v1/articles`                       | CRUD    |
| 個人簡介         | `/api/v1/teacher/profile`                | GET/PUT |

## 狀態與錯誤處理

| 狀態               | 處理方式                                   |
| ------------------ | ------------------------------------------ |
| Loading            | Skeleton Screen                            |
| 無已上架課程       | 空狀態 + 「建立第一堂課」CTA              |
| 影片上傳失敗       | 錯誤提示 + 重試按鈕                       |
| 課程審核中         | 標示審核狀態，禁止再次提交                 |
| 未授權（401）      | 導向登入頁面                               |

---

## 交互行為

| 行為             | 說明                                         |
| ---------------- | -------------------------------------------- |
| 影片上傳         | 拖拽上傳 + 上傳進度條                        |
| 統計圖表         | 互動式圖表，hover 顯示詳細數據               |
| 側邊導航         | 根據身份（學生/老師）動態顯示項目             |
| 響應式           | 儀表板圖表在行動端自適應寬度                 |
