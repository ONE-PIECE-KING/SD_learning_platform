# 學生中心 頁面規格書

## 頁面資訊

| 項目     | 說明                                           |
| -------- | ---------------------------------------------- |
| 路徑     | `/dashboard/*`                                 |
| 角色權限 | 需登入，角色為學生                             |
| 核心目的 | 學生學習主入口，管理課程、購課紀錄與個人設定   |

---

## 頁面結構

### 全局元件

**Header：**
- Logo（點擊返回首頁 `/`）
- 導航連結：課程總覽、一對一諮詢、資源分享
- 右側按鈕：會員中心（`/dashboard`）、通知鈴鐺（下拉面板，詳見 `11_notification_component.md`）、購物車（圖示 + 數字徽章）

**側邊導航欄（Nav）：**

| 項目         | 路徑                       |
| ------------ | -------------------------- |
| 我的課程     | `/dashboard/my-courses`    |
| 購課記錄     | `/dashboard/history`       |
| 一對一諮詢   | `/dashboard/consult`       |
| 設定         | `/dashboard/settings`      |
| 個人簡介     | `/dashboard/profile`       |
| 資源分享     | `/dashboard/resources`     |

## 元件規格

### 2.1 我的課程（`/dashboard/my-courses`）

**課程列表區（CourseListSection）：**

| 元件         | 說明                                                     |
| ------------ | -------------------------------------------------------- |
| 課程卡片     | 課程主圖、標題、簡介、課程進度條                         |
| 互動         | 點擊卡片跳轉至上課頁面                                   |
| 分頁按鈕     | 課程過多時提供分頁                                       |

**上課頁面（CoursePageSection）：**

佈局建議三欄式：左側章節列表、中間影片播放與互動區、右側筆記與聯絡簿。

| 元件               | 說明                                                             |
| ------------------ | ---------------------------------------------------------------- |
| 章節列表           | 固定側邊，標示已完成/正在觀看單元，點擊切換                     |
| 影片播放器         | 倍速播放、畫質調整、音量控制、全螢幕                             |
| 影片討論串         | 在特定影片時間點提問，問題與回覆顯示於影片下方                   |
| 筆記區             | 個人筆記、共筆、AI 摘要（進階功能）                               |
| 聯絡簿             | 作業查看/上傳、老師訊息、測驗                                     |

### 2.2 購課記錄（`/dashboard/history`）

| 元件           | 說明                                               |
| -------------- | -------------------------------------------------- |
| 篩選器         | 消費交易明細 / 退款交易明細                        |
| 列表           | 表格：日期、課程名稱、付款金額、訂單狀態           |

### 2.3 一對一諮詢（`/dashboard/consult`）

| 元件         | 說明                                                   |
| ------------ | ------------------------------------------------------ |
| 行事曆       | 瀏覽可預約老師及時段，進行預約/取消                    |
| 訪談室       | 視訊框、對話框、計時器、計費狀態                       |

### 2.4 設定（`/dashboard/settings`）

| 元件             | 說明                                         |
| ---------------- | -------------------------------------------- |
| 我要當老師       | CTA 區塊引導申請流程                         |
| Nav 編輯         | （選配）自訂側邊導航項目順序或隱藏           |

### 2.5 個人簡介（`/dashboard/profile`）

| 元件           | 說明                                   |
| -------------- | -------------------------------------- |
| 個人簡介表單   | 頭像、暱稱、自我介紹                   |
| 我的證書       | 完課電子證書展示                       |
| 我的作品       | 課程實作作品上傳與展示                 |

### 2.6 資源分享（`/dashboard/resources`）

| 元件         | 說明                                                     |
| ------------ | -------------------------------------------------------- |
| 筆記分享     | 將個人筆記公開分享給其他同學                             |
| 文章撰寫     | （選配，需權限控管）發布知識分享文章                     |

## API 整合

| 功能         | 端點                                     | 方法   |
| ------------ | ---------------------------------------- | ------ |
| 我的課程列表 | `/api/v1/student/courses`                | GET    |
| 課程詳情     | `/api/v1/courses/{id}`                   | GET    |
| 影片串流     | `/api/v1/videos/{id}/stream`             | GET    |
| 學習進度     | `/api/v1/student/courses/{id}/progress`  | GET    |
| 更新進度     | `/api/v1/student/courses/{id}/progress`  | PUT    |
| 購課記錄     | `/api/v1/student/orders`                 | GET    |
| 諮詢預約     | `/api/v1/consultations/book`             | POST   |
| 個人簡介     | `/api/v1/student/profile`                | GET/PUT|
| 筆記 CRUD    | `/api/v1/student/notes`                  | CRUD   |

---

## 狀態與錯誤處理

| 狀態           | 處理方式                                   |
| -------------- | ------------------------------------------ |
| Loading        | Skeleton Screen                            |
| 無已購課程     | 空狀態提示 + 「瀏覽課程」按鈕             |
| 影片載入失敗   | 錯誤提示 + 重試按鈕                       |
| 未授權（401）  | 導向登入頁面                               |

---

## 交互行為

| 行為           | 說明                                             |
| -------------- | ------------------------------------------------ |
| 章節列表       | 固定側邊，捲動時保持可見                         |
| 影片進度       | 自動記錄觀看進度，下次回來繼續播放               |
| 響應式         | 三欄 → 雙欄 → 單欄堆疊                          |
| 鍵盤快捷鍵     | 空白鍵暫停/播放，左右鍵快轉                     |
