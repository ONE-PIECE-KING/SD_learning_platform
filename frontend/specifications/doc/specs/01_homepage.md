# 首頁 頁面規格書

## 頁面資訊

| 項目     | 說明                                                         |
| -------- | ------------------------------------------------------------ |
| 路徑     | `/`                                                          |
| 角色權限 | 公開頁面，所有訪客皆可瀏覽                                   |
| 核心目的 | 品牌形象建立 → 價值傳達 → 名單沉澱（Email）→ 課程/服務轉換 |

## 頁面結構

首頁由 7 個主要區塊組成，由上至下依序排列：

| 順序 | 區塊名稱               | 用途                       |
| ---- | ---------------------- | -------------------------- |
| 1    | Hero                   | 品牌主視覺與核心 CTA       |
| 2    | 信任徽章條             | 建立專業信任感             |
| 3    | 主推課程               | 展示精選課程卡片           |
| 4    | 主要服務               | 介紹一對一諮詢與模擬面試   |
| 5    | 痛點與解決方案         | 引發共鳴並提供解方         |
| 6    | 社會證明               | 學員見證與成果展示         |
| 7    | Final CTA              | 最終轉換行動呼籲           |

## API 整合

### 課程卡片資料

```
GET /api/v1/courses?featured=true&limit=4
```

**回應結構：**

```json
{
  "id": "string",
  "title": "string",
  "thumbnail": "url",
  "description": "string (max 50 chars)",
  "price": "number",
  "original_price": "number | null",
  "instructor": "string",
  "rating": "number (1-5)",
  "rating_count": "number",
  "sort_order": "number"
}
```

### 見證卡片資料

```json
{
  "id": "string",
  "name": "string",
  "photo": "url",
  "previous_role": "string",
  "testimonial": "string (80-120 chars)",
  "achievement": "string",
  "rating": "number (1-5)"
}
```

## 狀態與錯誤處理

| 狀態     | 處理方式                                           |
| -------- | -------------------------------------------------- |
| Loading  | 課程卡片區域使用 Skeleton Screen                   |
| Empty    | 若無精選課程，隱藏 Section 3 或顯示預設占位        |
| Error    | API 失敗時顯示錯誤提示並提供重試按鈕               |

## 交互行為

### 轉換追蹤事件

| 事件名稱             | 觸發條件                     |
| -------------------- | ---------------------------- |
| page_view            | 頁面載入                     |
| scroll_depth         | 捲動至 25%/50%/75%/100%     |
| cta_click            | 點擊任何 CTA 按鈕           |
| lead_magnet_click    | 點擊「領取手冊」CTA         |
| course_card_click    | 點擊課程卡片                 |
| newsletter_subscribe | 完成 Email 訂閱             |

### 響應式行為

- Desktop：標準多欄佈局
- Tablet：卡片改為 2 欄
- Mobile：單欄堆疊，CTA 按鈕全寬

### 動畫

- 各區塊支援滾動進場動畫（fade-in / slide-up）
- CTA 按鈕 hover 時亮度 +10%
