# 資源分享 頁面規格書

## 頁面資訊

| 項目     | 說明                                           |
| -------- | ---------------------------------------------- |
| 路徑     | `/resources`                                   |
| 角色權限 | 公開頁面，所有訪客皆可瀏覽                     |
| 核心目的 | 文章部落格系統，提供免費學習資源與知識分享     |

---

## 頁面結構

| 區塊               | 說明                                         |
| ------------------ | -------------------------------------------- |
| 頂部搜尋與分類     | 搜尋框 + 分類標籤切換                        |
| 文章列表區域       | 文章卡片網格 / 列表                          |
| 分頁控制           | 頁碼導航                                     |

---

## 元件規格

### 搜尋與分類標籤

- **搜尋框：** 支援關鍵字搜尋文章標題、內容摘要或標籤
- **分類標籤：** 全部、技術新訊、知識分享、提示詞工程、轉職案例

### 分類說明

| 分類 | 內容方向 | 說明 |
| ---- | -------- | ---- |
| 技術新訊 | 業界技術動態 | 資料科學、AI、ML 等領域最新趨勢 |
| 知識分享 | 教學文章 / 教程 | 資料科學相關知識文章與教學 |
| 提示詞工程 | AI Prompt 技巧 | Prompt Engineering 教學與範例 |
| 轉職案例 | 學員成功故事 | 轉職資料科學的真實案例與心得 |

### 排序控制

下拉選單選項：最新發布、最多瀏覽、最多收藏

### 文章卡片（Article Card）

| 欄位         | 說明                                                     |
| ------------ | -------------------------------------------------------- |
| 封面圖       | 文章主圖（16:9 比例）                                   |
| 文章標題     | 最多 2 行，超出省略號                                   |
| 摘要         | 文章前 80-120 字摘錄                                     |
| 作者資訊     | 作者頭像 + 姓名（Caption 字級）                         |
| 發布日期     | 文章發布時間                                             |
| 分類標籤     | 所屬分類（技術新訊、知識分享等）                         |
| 技術標籤     | 技術關鍵字（如 #Python, #AI, #Pandas）                  |
| 瀏覽次數     | 熱門度指標                                               |

### 文章詳情頁（Article Detail）

| 區塊           | 說明                                                 |
| -------------- | ---------------------------------------------------- |
| 文章標題       | `<h1>` 層級                                          |
| 作者與日期     | 作者資訊 + 發布/更新日期                             |
| 文章內容       | Markdown 渲染，支援程式碼區塊語法高亮               |
| 分享按鈕       | 複製連結、分享至社群                                 |
| 相關文章推薦   | 同分類或同標籤的相關文章                             |

## API 整合

### 文章列表獲取

```
GET /api/v1/articles?search={keyword}&category={cat}&sort={sort}&page={n}&limit=12
```

- 載入文章列表，支援分頁

### 文章詳情

```
GET /api/v1/articles/{id}
```

- 載入單篇文章完整內容
- 同步增加 `view_count`

### 文章發布（講師/管理員）

```
POST /api/v1/articles
```

- **請求：** `{ title, content, category, tags, cover_image }`
- **權限：** 需講師或管理員身分

## 狀態與錯誤處理

| 狀態/錯誤       | 處理方式                                                   |
| --------------- | ---------------------------------------------------------- |
| Loading         | 文章卡片區域使用 Skeleton Screen                           |
| 搜尋無結果      | 顯示「目前尚無相關文章」，推薦「熱門文章」                 |
| 文章不存在      | 顯示 404 提示，推薦回到文章列表                            |
| API Error       | 顯示錯誤訊息 + 重試按鈕                                   |

## 交互行為

| 行為           | 說明                                                 |
| -------------- | ---------------------------------------------------- |
| 程式碼高亮     | 文章中的程式碼區塊使用語法高亮渲染                   |
| 分享功能       | 支援複製連結、社群分享                               |
