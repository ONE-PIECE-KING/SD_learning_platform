# 課程總覽 頁面規格書

## 頁面資訊

| 項目     | 說明                                   |
| -------- | -------------------------------------- |
| 路徑     | `/courses`                             |
| 角色權限 | 公開頁面，所有訪客皆可瀏覽             |
| 核心目的 | 展示所有課程，提供搜尋、篩選與排序功能 |

---

## 頁面結構

| 區塊               | 說明                                             |
| ------------------ | ------------------------------------------------ |
| 頂部搜尋區         | 搜索列 + 分類標籤/麵包屑導航                     |
| 側邊篩選欄         | 類別、難易度、價格區間、評分篩選                 |
| 主要內容區         | 排序控制 + 課程網格 + 分頁控制                   |

**響應式佈局：** Desktop 3-4 欄 / Mobile 1-2 欄

---

## 元件規格

### 搜索列（Search Bar）

- 輸入關鍵字即時搜尋課程
- 對應 API：`GET /api/v1/courses?search={keyword}`

### 側邊篩選欄（Filter Sidebar）

| 篩選項目 | 類型               | 說明                               |
| -------- | ------------------ | ---------------------------------- |
| 類別     | 多選/單選框        | 列出所有課程類別                   |
| 難易度   | 單選               | 初級、中級、高級                   |
| 價格區間 | 區間選擇           | 免費、付費、自定義價格區間         |
| 評分     | 單選               | 4 顆星以上、3 顆星以上             |

### 排序控制（Sort Controller）

下拉選單選項：最新上架、熱門程度、價格由高到低、價格由低到高

### 課程卡片（Course Card）

| 欄位       | 規格                                           |
| ---------- | ---------------------------------------------- |
| 課程縮圖   | 固定比例 16:9                                  |
| 課程標題   | 最多 2 行，超出省略號                          |
| 課程簡介   | 最多 50 字元                                   |
| 講師姓名   | Caption 字級，從 API 關聯資料提取              |
| 評價系統   | 星星圖標 + 數字評價                            |
| 價格顯示   | 優惠價（粗體）+ 原價（刪除線）；免費顯示「免費」|
| 行動按鈕   | 「查看詳情」→ `/courses/:id`、「加入購物車」   |

### 分頁控制（Pagination）

- 顯示總頁數、當前頁碼、上一頁/下一頁

## API 整合

### 初始資料獲取

```
GET /api/v1/courses
```

| 參數   | 預設值             | 說明         |
| ------ | ------------------ | ------------ |
| page   | 1                  | 當前頁碼     |
| limit  | 12                 | 每頁數量     |
| sort   | `created_at:desc`  | 排序方式     |

### 篩選與搜尋

- 使用者更改篩選條件或搜尋文字後，Debounce 處理並重新呼叫 API
- 篩選條件同步更新至瀏覽器 URL Query String，確保重新整理後篩選不失蹤

```
GET /api/v1/courses?search={keyword}&category={id}&level={level}&sort={criteria}&page={n}&limit=12
```

### 身份驗證與操作限制

- 已登入：檢查使用者是否已擁有課程（調整卡片顯示）
- 講師身分：顯示「建立課程」按鈕 → `POST /api/v1/courses` 表單頁

## 狀態與錯誤處理

| 狀態                | 處理方式                                                 |
| ------------------- | -------------------------------------------------------- |
| Loading             | 使用 Skeleton Screen 填充課程卡片區域                    |
| 搜尋無結果          | 顯示「找不到符合條件的課程」+ 「清除所有篩選條件」按鈕  |
| API Error (500/404) | 顯示錯誤訊息元件 + 「重新整理」按鈕                     |
